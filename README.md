REST API INTEGARTIO PROJECT – COUNTRY SEARCH APP

Project Description
This project is a Country Search Application developed using HTML, CSS, and JavaScript. It uses the Fetch API to retrieve country data from a public REST API and display it dynamically based on the user's search.

API USED
API Name: Countries Now API

API Endpoint
https://countriesnow.space/api/v0.1/countries/flag/images

Technologies Used
•	HTML5 
•	CSS3 
•	JavaScript 
•	Fetch API


Features
•	Search for a country by name. 
•	Display country information dynamically. 
•	Loading message while data is being fetched. 
•	Error message when a country is not found. 
•	Responsive design.


How the API Works
When the user enters a country name, JavaScript sends a request to the API using the Fetch API. The application receives the response, searches for the entered country, and displays its information on the webpage.
